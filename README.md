# UMCONVO App
Figma Link: https://www.figma.com/design/atbTwU5vZn51uMa1rqnw7m/UMConvo-App?node-id=0-1&t=obyz6UAnpHvj57aL-1
## Final Year Project
## Developer: LIM KEE EN
UMConvo is a mobile application designed to enhance the convocation experience for Universiti Malaya students and staff. The app provides easy access to convocation schedules, announcements, and related services, offering a modern alternative to the existing web-based system.

<img width="418" height="451" alt="image" src="https://github.com/user-attachments/assets/d39bbb08-d8a9-49e1-aedc-61fa7da937bf" />

## Features
1. Convocation Schedule: View detailed schedules for convocation ceremonies.
2. Announcements & Notifications: Receive updates on important events and changes.
3. Event Reminders: Set reminders for convocation-related events.
4. User Profiles: Students can view their personal convocation details.
5. Social Feed: Upload post to create memory for their graduation.

## Technology Stack
* Frontend: React, React Native, Tailwind CSS
* Backend: Node.js, Firebase, REST API
* Database: Firebase Realtime Database, Firestore
* Design Tools: Figma

## Setup Instructions
```
# Clone repo
git clone https://github.com/LimKeeEn/umconvo-app.git
npx expo start
```

## Backend
```
cd backend
npm install
npm run dev
```

## Admin
```
cd admin
npm start
```
